Interleaved Review

No new material — just retrieval. Questions are deliberately shuffled across all six lessons, so you practise recognising which idea applies, not just recalling a single topic in order.

This is the lesson that makes the rest stick. Reading feels like learning but fades fast (fluency); pulling answers from memory, cold and out of order, builds the durable kind (storage strength). Don't peek back first — attempt each, then read the feedback. Come back to this page again in a few days; spacing is the other half of retention.

How to use this Answer from memory before clicking. A wrong answer with effort beats a right answer you guessed — the struggle is what consolidates it. Re-take the whole set next session.
