Scan, zones, priority scan, dual-watch — local codeplug behaviours. They never traverse the air; each radio implements them in firmware/CPS. Program per radio; no cross-vendor concern.

Bridge rule of thumb Anchor the product on the safe pile (voice + call types + emergency flag) so it works against any compliant fleet. Scope everything in the single-vendor pile as per-customer integration — and never claim cross-vendor encryption without bench-testing the exact radios.
